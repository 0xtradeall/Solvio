5. NETWORK DETECTION & AUTO-SWITCH

When a payer opens a payment link (/pay):
- Detect which network the payer's Phantom wallet 
  is connected to
- If it doesn't match the app's network (devnet for 
  now, mainnet in production):
  * Show a yellow warning banner at the top:
    "⚠️ Wrong network detected. Solvio requires 
    Solana Devnet."
  * Show a button: "Switch to Devnet"
  * On click: trigger Phantom's network switch request
  * Once switched: banner disappears, payment form shows
- Handle the case where Phantom doesn't support 
  programmatic network switching gracefully:
  * Show manual instructions: 
    "Open Phantom → Settings → Developer Settings → 
    Switch to Devnet"
